Technical Checklist

1. Login Forms
  a. One twig template to generate every type of form used
  b. Mysql database to save and validate requests
  c. Less to generate css styles
  d. Sucessful login changes PHP session variable
  e. Ajax to manage requests and responces.

2. Home page
   a. Twig templates for header, footer, and different parts of body like comics cards, about section
   b. Mysql database to get information about comics
   c. PHP session variables to store information like user name and id, and navbar options
   d. Ajax to add items to cart.

3. Cart page
   a. Twig table template to display every item added to cart.
   b. PHP session variables to display store and display items in cart.
